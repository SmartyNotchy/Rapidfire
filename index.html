Settings

Checker Strictness

Sets how close your input has to be to the actual answer for it to be accepted by the SAQ checker. Lower settings are more forgiving with typos but may incorrectly accept some inputs.

Show Question Topic

Display which topic set each question is drawn from.

Skip Shortcut

If enabled, double-tapping ENTER while no answer is selected will automatically skip the problem.

MCQ Keybinds

If enabled, pressing the letter keys (ABCD, etc.) corresponding to an MCQ option will automatically select it.

Redo Skipped Questions

Add skipped questions to the back of the question queue.

Redo Failed Questions

Add failed questions (not answering correctly on the first attempt) to the back of the question queue.

Insert Do-Overs Randomly

When a question is redone, add it in a random position in the question queue instead of at the back.
